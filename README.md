# 1001 Albums Project

## Project Summary
*1001 Albums You Must Hear Before You Die* is a book by Robert Dimery that takes some of the most influential and/or best albums released since the 1950's and makes a case for why they are important, using the reviews of music critics. For my project, I will create a music recommendation system based on these albums where users can input an artist, album, song, or genre from the list and get some recommendations for specific songs based on that.
